# 💻 DevTycoon

**DevTycoon** est un jeu de type *Idle Clicker* développé sous Android avec Jetpack Compose. Incarnez un développeur indépendant, gérez votre code, améliorez votre matériel et gravissez les échelons du monde de la tech.

## 🛠️ Fonctionnalités
- **Moteur de clic :** Génération de LOC (lignes de code) via des clics tactiles.
- **Système de Marché :** Améliorez votre matériel (claviers, serveurs) et vos composants logiciels.
- **Système de Quêtes :** Objectifs dynamiques pour booster votre progression.
- **Arbre de talents :** Investissez vos points de talent pour automatiser votre production.
